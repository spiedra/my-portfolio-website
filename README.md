# My portfolio
My personal website 

## Tech Stack
- Next.js - React Framework for Production
- Styled Components
- ESLint

## Getting Started

Prerequisites

- git
- node.js

Installation

1. Clone the repository

```bash
git clone https://github.com/spiedra/my-portfolio-website
```

2. Install npm packages

```bash
$ npm install
```

3. Run the project

```bash
$ npm run dev
```
